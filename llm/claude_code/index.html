安装 🔗

几种方案：
1. `PowerShell 原生安装`（官方推荐）
irm https://claude.ai/install.ps1 | iex


2. `CMD 原生安装`
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd


3. `WinGet 安装`
winget install Anthropic.ClaudeCode


4. `npm 全局安装`
npm install -g @anthropic-ai/claude-code
这个方案需要 `Node.js 18+`。官方对 Windows 也建议配 `Git for Windows`。

5. `WSL 里安装`
curl -fsSL https://claude.ai/install.sh | bash
如果你平时就在 WSL 里开发，这通常是 Windows 上兼容性最好的一种。

权限模式 🔗

Claude Code 的权限模式（permission mode）决定哪些操作会自动执行、哪些需要确认。下表汇总各模式的作用与开启方式。

模式 作用 如何开启
default 默认模式；基本是只读操作直接执行，其他操作会询问 * 直接启动就是这个模式
* 或：claude --permission-mode default
* 或在 settings.json 里设："defaultMode": "default"
acceptEdits 自动接受文件编辑和常见文件系统命令 * 会话中按 Shift+Tab 切换到它
* 或：claude --permission-mode acceptEdits
* 或设："defaultMode": "acceptEdits"
plan 只分析、读文件、写计划，不改代码 * 会话中按 Shift+Tab 切换到它
* 或：claude --permission-mode plan
* 或设："defaultMode": "plan"
* 也可在单条消息前用 /plan
auto 自动执行大多数操作，但带后台安全检查 * claude --permission-mode auto
* 或设："defaultMode": "auto"
* 满足条件时，也可能通过 Shift+Tab 切到
dontAsk 不会弹确认框；只有预先批准的工具才能用 * claude --permission-mode dontAsk
* 或设："defaultMode": "dontAsk"
* 不会出现在 Shift+Tab 循环里
bypassPermissions 跳过几乎所有权限检查，最激进 * claude --permission-mode bypassPermissions
* 或：claude --dangerously-skip-permissions
* 或设："defaultMode": "bypassPermissions"

说明： 可通过命令行 --permission-mode、配置文件里的 defaultMode，以及会话内 Shift+Tab（部分模式）或消息前缀（如 /plan）切换；dontAsk 不在 Shift+Tab 循环中；bypassPermissions 另有危险标志 --dangerously-skip-permissions，使用前请自行评估风险。按shtift+tab可以进行模式切换

上下文和记忆、会话 🔗

  • /resume: 会话恢复，claude -c 恢复上次的会话
  • /context：查看当前占用上下
  • /compact： 当你完成一个阶段性任务后，前面的内容“还有一点价值，但已经不需要保留全部细节”时，建议手动执行
  • /clear: 清空会话所有
  • /init：生成 cloud.md 文件（项目级配置）
  • /memory 可以选择项目级别或者用户全局

安装MCP 🔗

网址：https://mcpmarket.com/

  • 安装 Playwright MCP（用途是可以控制浏览器）
claude mcp add playwright npx @playwright/mcp@latest

如果你能看到 playwright，说明它已经安装成功。但可以看到有一个 “failed”，表示有问题。不用自己排查。 这正是 Claude Code 的强项：直接让它自己解决。

  • 安装 HyperFrames Skill，让 AI 制作视频
帮我安装 npx skills add /-com/hyperframes

安装skills 🔗

网址：https://skills.sh/ 存放目录：~/.claude/skills

终端交互 🔗

  • ! 号，进入bash模式，进入终端模式，可执行任意 shell 命令（如open index.html、mkdir、npm install）
  • 当运行是阻塞进程时：输入ctrl+b 可以让任务进入后台
  • 长文本：文本换行shift+enter，macos:option+enter，其他编辑器打开，ctrl+g，默认是打开vi，设置默认文件打开软件如vscode
  • 查看当前后台任务 /tasks，输入k 结束后台任务
  • ctrl+c复制文件，ctrl+v粘贴到会话或者文件直接拖入会话
